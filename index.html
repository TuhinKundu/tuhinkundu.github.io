Selected projects
Cancerous blood cell segmentation with U-Net architecture with GUI
Detection and segmentation of cancerous blood cell segmentation using CNN based deep learning model with a GUI tool for doctors to use. The data consisted of stained image slides which were obtained from a cancer research institute which was incredibly noisy and 100x microscopic in nature. The pipeline included preprocessing, annotation, trained model and a GUI. The model was trained using U-Net architecture with multi-channel loss, where one channel accounted for cell boundary region and another for nuclei and cytoplasm. The trained weights were fitted onto a python GUI built using tkinter and results were benchmarked for future reference. Data and code publicly not available.
Image captioning using encoder-decoder architecture
Image captioning model with CNNs as encoder and RNNs as decoder. Resnet-101 is the encoder in the model, decoder unit could be chosen between LSTM, Glove or BERT model. Empirical analysis performed to determine performance trade-off with BLEU being used as an evaluation metric.

Keyphrase extraction using unsupervised PageRank model with position and theme bias
Unsupervised keyphrase extraction model using PageRank graph from research articles. PositionRank and Key2vec were used to rank the keyphrases incorporating frequency and position of candidate keyphrases and theme of the research articles using fasttext embeddings.
